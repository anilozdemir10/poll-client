# Poll Application

Poll Application is an application where you can add as many options as you want and vote them.

## Client

```bash
Next JS - http://localhost:3000
```

## Installation

```bash
yarn install
```

## Getting Started

Development:

```bash
yarn dev
```

Build:

```bash
yarn build
```

Production:

```bash
yarn start
```

Prettier:

```bash
yarn prettier
```
